# 🎓 Smart Campus Complaint System

A web-based complaint management platform designed to streamline the process of reporting, tracking, and resolving campus-related issues. The system provides a centralized platform for students to submit complaints and administrators to manage and resolve them efficiently.

## 🚀 Features

### Student Module

* User Registration & Login
* Submit Complaints
* Track Complaint Status
* View Complaint History
* Receive Status Updates

### Admin Module

* Secure Admin Login
* View and Manage Complaints
* Update Complaint Status
* Categorize Complaints
* Generate Reports and Analytics

### System Features

* Real-Time Complaint Tracking
* User-Friendly Interface
* Secure Authentication
* Centralized Complaint Database
* Responsive Design

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/unnatisuple/smart-campus-complaint-system-main.git
cd smart-campus-complaint-system-main
```

### Install Dependencies

Frontend:

```bash
cd client
npm install
```

Backend:

```bash
cd server
npm install
```

### Run Application

Backend:

```bash
npm start
```

Frontend:

```bash
npm run dev
```

## 📌 Workflow

1. Student logs in.
2. Student submits a complaint.
3. Complaint is stored in the database.
4. Admin reviews the complaint.
5. Admin updates the complaint status.
6. Student tracks progress until resolution.

## 🎯 Objectives

* Improve communication between students and administration.
* Reduce manual complaint handling.
* Increase transparency in issue resolution.
* Provide a centralized complaint management platform.
